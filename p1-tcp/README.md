## 1.4 流量控制
  由于客户端和服务端处理数据的能力不同，所以数据发送方和数据接收方都有固定的缓冲区大小。
  发送数据时，最多只能发送接收端缓存区允许的大小，从而避免缓冲区溢出问题。
## 1.5 全双工
  客户端和服务器可以同时向对方发送数据。
